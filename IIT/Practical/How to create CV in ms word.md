Procedure:-
- Open **Word → Blank Document**.
- Set **Layout → Margins → Normal**, **Orientation → Portrait**.
- Type **Name (bold, larger font)** and your **contact info**.
- Add **Education** → list degree, institution, year.
- Add **Work Experience** → job title, company, dates, bullets for responsibilities.
- Add **Skills** → list key skills.
- (Optional) Add **Projects/Achievements**.
- Check **formatting and spacing**.
- Save: **File → Save As → Word or PDF**.
